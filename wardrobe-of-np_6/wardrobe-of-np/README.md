# Wardrobe of NP ✦

Personal wardrobe planner for Nilomi & Parth.

## Quick Start

See **SETUP_GUIDE.md** for full instructions.

## Files

- `index.html` — the entire app (open this in a browser, or deploy to Netlify)
- `netlify.toml` — Netlify deployment config
- `SETUP_GUIDE.md` — step-by-step Google Drive + Netlify setup

## Features

- 🌸 Nilomi's wardrobe (maroon theme) + 👔 Parth's wardrobe (navy theme)
- Occasions, events, looks with multiple photos (swipeable)
- Jewellery, shoes, accessories per look — manual or from Closet
- My Closet — master list of all items, assign to any occasion
- Prep checklist per look
- Comments & notes
- Save custom outfit combos
- Print PDF — Nilomi only, Parth only, or combined NP booklet
- Dark mode
- Google Drive sync — sign in once, data loads on any device
- Auto-saves every 1.5 seconds after changes
